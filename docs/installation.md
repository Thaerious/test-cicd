# Installation

## Stable release

To install test-cicd, run this command in your terminal:

```sh
uv add test-cicd
```

Or if you prefer to use `pip`:

```sh
pip install test-cicd
```

## From source

The source files for test-cicd can be downloaded from the [Github repo](https://github.com/Thaerious/test_cicd).

You can either clone the public repository:

```sh
git clone git://github.com/Thaerious/test_cicd
```

Or download the [tarball](https://github.com/Thaerious/test_cicd/tarball/master):

```sh
curl -OJL https://github.com/Thaerious/test_cicd/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd test_cicd
uv pip install .
```
